# RPG Console Battle

Juego RPG por turnos en consola desarrollado en Python usando
Programación Orientada a Objetos (POO).

## Características
- Combate por turnos
- Personajes: Héroe y Orco
- Armas aleatorias
- Golpes críticos y fallos
- Sistema simple de resistencia (vida)

## Cómo ejecutar el juego
Asegúrate de tener Python instalado y ejecuta:

```bash
python RPG.py
